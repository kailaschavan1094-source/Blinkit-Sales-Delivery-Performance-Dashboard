# Blinkit Sales & Delivery Performance Dashboard

Interactive Power BI dashboard built to analyze Blinkit sales, delivery performance, customer feedback, and product insights using Power BI, Power Query, and DAX.

![Dashboard](Dashboard-Screenshot.png)

---

## Project Overview

This project provides an interactive dashboard to monitor business performance across multiple dimensions.

Key business questions answered:
- What is the total revenue generated?
- How many orders and customers are there?
- Which categories generate the highest revenue?
- What is the average delivery time?
- What is the customer feedback sentiment?
- Which products and brands perform best?
- How is delivery status distributed?
